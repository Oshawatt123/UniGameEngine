\doxysubsection{Detailed Description}
The Resouce Manager is responsible for almost all of the resources used by the game. It prevents the loading of textures that have already been loaded. It also loads Maps and Scenes. Loading the same scene twice is handled by \mbox{\hyperlink{class_scene_manager}{Scene\+Manager}}. Audio is handled by the \mbox{\hyperlink{class_audio_manager}{Audio\+Manager}}.
