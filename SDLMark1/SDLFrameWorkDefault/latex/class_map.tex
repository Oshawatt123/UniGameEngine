\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_map}{Map}} class is responsible for holding data about a \char`\"{}map\char`\"{}. It will parse a text file that could look like the following \begin{DoxyVerb}ABBBBBBBBBC
DJJJLJJJJJF
DJLJJLKJJJF
DJJKJLJJJJF
DJJJJJJJKJF
DJKLJKJJLJF
DJJJJLJJJJF
GHHHHHHHHHI
\end{DoxyVerb}


Each line is stored as a string and all the lines are kept in a vector. Each character is used to index into a tile\+Set, and the corresponding tile is drawn into the world.
