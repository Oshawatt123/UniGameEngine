\doxysubsection{Detailed Description}
An \mbox{\hyperlink{class_entity}{Entity}} is the base of every object inside Fithy\+Engine.

An entity is comprised of multiple Components, as well as some identifying features such as an ID, and name.

E\+V\+E\+RY entity will, by default, have a \mbox{\hyperlink{class_position_component}{Position\+Component}}.
