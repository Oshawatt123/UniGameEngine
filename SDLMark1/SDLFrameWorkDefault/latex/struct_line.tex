\doxysubsection{Detailed Description}
A struct to help with drawing debug lines in Filthy\+Engine. A \mbox{\hyperlink{struct_line}{Line}} is defined by a 2 \mbox{\hyperlink{class_vector2}{Vector2}}\textquotesingle{}s for the start and end of the lines, treated as position co-\/ordinates in world-\/space as well as a color.
