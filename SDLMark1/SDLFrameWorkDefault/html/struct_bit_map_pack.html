Detailed Description

BitMapPack is a storage class for textures, and their attributes. It exists to make passing textures around easier, as it eliminates the need to also pass the width and height as extra parameters just taking up space. This also exists because SDL_Texture does not contain a definition for width and height. That data is stored on the SDL_Surface, which is deleted as soon as the asset is loaded. Hence making it a wise idea to grab this data when the image is first loaded and the surface generated.
