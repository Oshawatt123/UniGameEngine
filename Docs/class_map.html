Detailed Description

The Map class is responsible for holding data about a "map". It will parse a text file that could look like the following 

ABBBBBBBBBC
DJJJLJJJJJF
DJLJJLKJJJF
DJJKJLJJJJF
DJJJJJJJKJF
DJKLJKJJLJF
DJJJJLJJJJF
GHHHHHHHHHI

Each line is stored as a string and all the lines are kept in a vector. Each character is used to index into a tileSet, and the corresponding tile is drawn into the world.
